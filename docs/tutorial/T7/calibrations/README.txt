Contains example YAML files and scripts to run calibrations with them

- `T7_YAML_*` files correspond to the examples of YAML calibrations used in Tutorial 7
- `WE_*.yaml` are the solutions to the 'worked example' section in Tutorial 7
- `typ_calibration_instructions.yaml` corresponds to a full finished YAML calibration for the model

This folder also contains

- `*.py` files to run the YAML files and produce `*.xlsx` calibration outputs
- `*.ipynb` notebooks to generate plots using those calibration outputs
